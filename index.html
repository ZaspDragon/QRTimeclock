Setup order

  1. Add your Firebase config in firebase-config.js.
  2. Create manager/admin users in Firebase Authentication.
  3. Create matching Firestore docs in users for managers/admins.
  4. Publish Firestore rules and create any prompted composite indexes.
  5. Deploy and print the company QR poster.
